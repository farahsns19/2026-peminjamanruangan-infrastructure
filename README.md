# Sistem Peminjaman Ruangan Kampus - Infrastructure

Repositori ini berisi dokumentasi dan catatan infrastructure untuk project Sistem Peminjaman Ruangan Kampus.

---

## Deskripsi

Infrastructure repository digunakan untuk menyimpan informasi terkait:

- Database yang digunakan
- Rencana deployment aplikasi
- Catatan environment sistem

Repo ini dibuat sebagai bagian dari tugas Persiapan PDBL 2026.

---

## File Dokumentasi

- `database.md` → menjelaskan struktur database dan tabel utama
- `deployment.md` → panduan menjalankan aplikasi dan rencana deployment
