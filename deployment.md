# Dokumentasi Deployment

Dokumen ini berisi catatan rencana deployment untuk Sistem Peminjaman Ruangan Kampus.

---

## Arsitektur Sistem

Sistem terdiri dari:

- Backend: ASP.NET Core Web API
- Frontend: React + Vite
- Mobile: Flutter (planned)
- Database: SQLite

---

## Cara Menjalankan Secara Lokal

### Backend

```bash
cd RoomBooking.API
dotnet run
```
Backend berjalan di http://localhost:5133/swagger

### Frontend
npm install
npm run dev
Frontend berjalan di:
http://localhost:5173
